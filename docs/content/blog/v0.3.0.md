# v0.3.0 — First stable release

The first stable release of Ironic ships with the complete foundation:

## Core features (v0.3.0+)

- **Module graph compiler** — validates providers, controllers, imports, and exports at startup
- **Dependency injection container** — singleton and transient scopes with `OnceCell` and optional resolution
- **Controller routing** — `#[controller]` + `#[get]`/`#[post]`/`#[put]`/`#[delete]` macros
- **Request pipeline** — Middleware → Guards → Interceptors → Extraction → Pipes → Handler
- **CLI generator** — `ironic new`, `ironic generate module|controller|service|resource|middleware|guard`
- **TestApplication** — socket-free, in-process HTTP testing with mock override support
- **Lifecycle hooks** — `OnModuleInit`, `OnApplicationBootstrap`, `OnModuleDestroy`, `OnApplicationShutdown`
- **Axum platform adapter** — compression, body limit, timeout, `configure_router()` escape hatch
- **Configuration** — typed layered config with environment variable overrides and redacted secrets
