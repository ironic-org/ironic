# v0.3.6

## Better project scaffolding

New projects now include `serde`, `serde_json`, `garde`, and `dotenvy` as dependencies. The generated Cargo.toml lists all available optional features with comments so you know exactly what's available.

## What's new

- **Validation pipes documentation** — expanded with comprehensive examples and garde rule reference
- **Auth API example** — full authentication example with JWT, sessions, and OAuth strategies
- **Basic API example** — CRUD example showing controllers, services, DTOs, and validation
- **Project manifest** — `features = ["security", "compression", "metrics", "validation"]` enabled by default
- **Authentication module** — dead code warning fix for generated test files
