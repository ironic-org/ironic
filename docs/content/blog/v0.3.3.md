# v0.3.3

## Smarter code generation

`ironic generate` now automatically adds required Cargo dependencies when scaffolding new modules. No more manual `Cargo.toml` edits after generating a resource.

## What's new

- **Auto-dependency injection** — CLI adds required crates to `Cargo.toml` on module registration
- Improved manual instruction formatting during generation
