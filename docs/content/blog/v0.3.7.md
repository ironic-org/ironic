# v0.3.7

## You can now wire middleware from main.rs

The biggest missing piece of the builder API is here:

```rust
Application::builder()
    .module(AppModule::definition())
    .middleware(MyCustomMiddleware)
    .middleware(AnotherMiddleware)
    .platform(AxumAdapter::new()...)
    .build()
    .await?;
```

Previously, global middleware could only be registered via `CompiledHttpApplication` internally or per-controller/per-route. Now it's a first-class builder method.

## What's new

- **`.middleware()` on `ApplicationBuilder`** — register any `impl Middleware` at the app level
- **`CompiledHttpApplication::extend_middleware()`** — internal API to batch-apply pre-erased middleware
