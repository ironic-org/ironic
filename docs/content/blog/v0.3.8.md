# v0.3.8

## Global middleware for every new project

`ironic new` now generates `main.rs` with a full production middleware stack:

```rust
Application::builder()
    .middleware(SecurityHeadersMiddleware::new(SecurityHeadersConfig::default()))
    .middleware(RateLimitMiddleware::new(rate_limit_max, 60))
    .middleware(CorsMiddleware::new(CorsConfig::new().allowed_origins(cors_origins)))
    .platform(AxumAdapter::new().compression()...)
    .build().await?;
```

**Stack order:** SecurityHeaders → RateLimit → CORS → Metrics → Compression → Body Limit → Timeout

## What's new

- **Global middleware API** — `Application::builder().middleware()` supports any `impl Middleware`
- **Security headers by default** — HSTS, CSP, X-Frame-Options, and 6 more headers enabled automatically
- **CORS via JSON array** — `CORS_ORIGINS=["https://app.com"]` in `.env` using `serde_json`
- **Rate limiting** — 100 req/min/IP by default, configurable via `RATE_LIMIT_MAX`
- **Fixed Dockerfile** — binary name matches project, no directory overwrite, `SERVER_HOST=0.0.0.0`
- **ironic-security re-exports** — `use ironic::security::{CorsMiddleware, RateLimitMiddleware, SecurityHeadersMiddleware}`
